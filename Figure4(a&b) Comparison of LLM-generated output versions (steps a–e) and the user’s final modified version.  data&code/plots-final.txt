地狱的召唤
杨玉环思念人间春节热闹，去找阎王哭诉想回到人间，吕后在旁边听到他们的对话，说自己也有很大的遗憾也想要回去。阎王不堪其扰，为了公平，宣布因地狱新公告——“最悲惨最遗憾的人可以重返人间”。
命运的陈述
五位女性开始逐一陈述自己的命运。武则天以冷静的语气回顾了她如何从后宫中脱颖而出，成为中国历史上唯一的女皇帝，但也坦言权力的代价让她失去了家庭与真情。吕后则带着隐隐的愤怒讲述自己如何被后世污名化，她强调自己的一切手段都是为了生存与保护家族。林徽因理性地分析了自己在男性主导的社会中追求独立的艰难，她感慨女性的才华常被忽视。何香凝则以坚定的语气讲述自己在革命中牺牲家庭的痛苦，她呼吁对女性贡献的重新审视。其中，杨玉环贯穿全场，哀婉地诉说自己的爱情悲剧，认为自己只是历史政治博弈中的牺牲品。
冲突与反思
陈述过后，五位女性间的争论逐渐升级。武则天与吕后因权力手段的正当性发生激烈争执，杨玉环试图用自己的牺牲打动她们，但被林徽因冷静地指出历史的不公。何香凝则愤然表示，争论谁更悲惨毫无意义，她们都只是被历史选择的牺牲者。这一言引发了短暂的沉默，众人开始反思自己的命运与女性的历史地位。阎王静静旁观，眼中闪过一丝深意。
被隐没的她
在短暂的沉默后，何香凝意识到在回到当时的传统男权主义社会下，大家的遗憾其实都不可能被弥补，争论毫无意义，引导历史上的女性角色们开始认识自己的价值和贡献。林徽因提出女性不应再争论谁更悲惨，从理性角度分析女性对社会的贡献，倡导女性应该团结起来发声。武则天和吕后放下成见，开始回顾自己作为女性的成就与牺牲。武则天用坚定的语气指出规则的局限性，吕后则以自身的经历阐释历史对女性的偏见。杨玉环则用哀婉的声音诉说她对人间的思念诉说女性在历史中的牺牲，但更希望能够为后世女性带来改变。五人达成共识，决定联手说服阎王，打破规则，一同重返人间。
山河换新颜
五位女性在阎王的许可下重返人间，她们的身影逐渐融入晨光中。阎王看着她们离去，露出一丝意味深长的微笑，仿佛期待她们在人间的新篇章。在现代，武则天成为为人民服务的政府领袖，杨玉环成为新生代实力爱豆，林徽因获得建筑业最高荣誉普里兹克建筑奖，而何香凝成为优秀的女性题材的电影编剧。
